üì• Import from Excel

Upload your existing Excel file to import all transactions at once.
Expected format: Description | Month | Amount | Category | Status

üìã Import Instructions

  1. Your Excel file should have columns: Description, Month, Amount, Category, Status
  2. Month can be text (e.g., "January") or a date
  3. Amount can include $ sign, commas, or be negative (will be converted)
  4. Categories will be automatically added if they don't exist
  5. Status should be either "Done" or "Scheduled"

üí° Import Tips

  • Backup First: Always keep a backup of your original Excel file
  • Data Validation: Ensure dates and amounts are formatted correctly
  • Categories: New categories will be created automatically
  • Large Files: Import may take a few moments for files with many transactions

üìÇ Data Files Information

Your expense tracker stores data in JSON files on your computer.

Data Files

  • transactions.json All your transaction records
  • budgets.json Your budget settings
  • categories.json Custom categories

Configuration File

~/.expense_tracker_config.json

This file stores your application settings (like data storage path) and persists across app updates.

Backup Your Data

To backup your data, simply copy all files from your data folder. You can restore by copying them back. Consider setting your data path to a cloud-synced folder (Dropbox, Google Drive) for automatic backups.

üí° Pro Tip:

Use the Export Excel/CSV feature regularly to create additional backups. These exports can be re-imported if needed, providing an extra layer of data protection.
